# FitGuard — 26 Ready-to-Paste Image Prompts

**How to use (per image):**
1. Open Magnific/Freepik Image Generator → settings: **model Auto · aspect 1:1 · 4 variations**.
2. Copy the whole prompt block, paste, Generate.
3. Pick the cleanest **side-view** result → download.
4. Rename the file to the **filename** shown → drop it in this folder (`public/exercises/`).
5. When done: `npm run build && npx cap sync ios` — they appear automatically.

> Format: `.webp` preferred, or `.png`/`.jpg`. Square. Aim < ~150 KB each.
> Every prompt already includes the locked style, so the whole set stays consistent.
