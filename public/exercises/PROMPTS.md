# FitGuard — All Exercise Prompts (START + END, exaggerated contrast)

Each exercise = 2 images the app crossfades so the rep MOVES:
- **START** → save as `<id>.png`
- **END**   → save as `<id>_2.png`

Every prompt below is COMPLETE — copy the whole block, paste, generate, download,
rename to the filename, drop in this folder (`public/exercises/`).

### 2 rules that stop the "both poses look the same" problem
1. **Use START as a reference for END.** In Magnific, drag the START image into the
   references box, then paste the END prompt. Keeps the same man, changes only the pose.
2. The prompts below are written at the TWO EXTREME ENDS of each rep on purpose
   (full stretch vs full contraction). Don't soften them.

Settings: model Auto · 1:1 · 4 variations. Save square, ~800px.
