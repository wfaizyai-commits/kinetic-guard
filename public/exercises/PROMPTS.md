# FitGuard — 25 Ready-to-Paste Image Prompts

Every prompt below is **complete and standalone** — no editing needed. Copy a whole
block, paste into Magnific/Freepik, generate, download the cleanest result, rename to
the **filename**, drop it in this folder (`public/exercises/`).

**Settings:** model Auto · aspect 1:1 · 4 variations.
**Format:** `.webp` preferred (or `.png`/`.jpg`). Square. Aim < ~150 KB each.
When done: `npm run build && npx cap sync ios` — images appear automatically.

> Style is identical in every prompt so the whole set stays consistent.
